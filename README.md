# Privacy-Preserving Causal Digital Twins for Pharmaceutical Logistics

This repository contains the simulation, machine learning, and Web3 execution code supporting the manuscript: **"Decentralized Pharmaceutical Logistics: Mitigating Spoilage via Causal Graph Neural Networks and Zero-Knowledge Execution."**

## Repository Structure
* `main_simulation_and_analysis.ipynb`: The primary Jupyter Notebook containing the 5-step pipeline:
  1. 50-Node Network Topology Initialization (Barabási-Albert)
  2. Memory-Optimized 30-Day Digital Twin Simulation
  3. AI Engine Initialization (PyG + MAPIE v0.8.6)
  4. Causal Inference Engine (DoWhy)
  5. Web3 Autonomous Agent Execution (ERC-4337 Proxy)
* `requirements.txt`: The exact Python environment dependencies required to reproduce the study.

## Execution Instructions
1. Clone this repository.
2. Install the required dependencies: `pip install -r requirements.txt`
3. Download the `pharma_telemetry_30d.parquet` dataset from the Zenodo link above and place it in the root directory.
4. Execute the Jupyter Notebook sequentially to reproduce the 74.00% GNN accuracy, the 97.93% ATE causal proof, and the Web3 gas optimizations.
